## Features Released This Wave

| Feature | Description |
|---------|-------------|
| Reward pool funding | Creator funds poll with tokens |
| MANUAL_PULL claims | Voters claim their own rewards |
| MANUAL_PUSH distribution | Creator distributes to all voters |
| Platform fee (2%) | Fee collected on distributions |
| Reward tracking UI | View earned/claimed rewards |
| Multi-token support | PULSE, MOVE, USDC rewards |
| RewardTicket records | On-chain proof of reward eligibility |

**Cumulative:** All Wave 1-2 features + Wave 3 features
