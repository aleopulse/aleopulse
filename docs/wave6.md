## Features Released This Wave

| Feature | Description |
|---------|-------------|
| Staking contract | `leopulse_staking.aleo` deployment |
| Lock periods | 7, 14, 21, 30, 90, 180, 365 days |
| Bronze tier | 0+ PULSE = 3 daily votes |
| Silver tier | 1,000+ PULSE = 6 daily votes |
| Gold tier | 10,000+ PULSE = 9 daily votes |
| Platinum tier | 100,000+ PULSE = 12 daily votes |
| StakeReceipt | On-chain staking proof |
| Staking UI | Stake/unstake interface |
| Vote limits | Enforce daily limits by tier |

**Cumulative:** All Wave 1-5 features + Wave 6 features
