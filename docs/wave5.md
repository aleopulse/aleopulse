## Features Released This Wave

| Feature | Description |
|---------|-------------|
| Private visibility mode | Polls visible only to invitees |
| PollInvite records | On-chain invite proof |
| Invite generation | Create invites for users |
| Invite links | Shareable invite URLs |
| Invite expiration | Time-limited access |
| Invite management UI | Dashboard for managing invites |
| Bulk invite generation | Create many invites at once |
| Access validation | Verify invite before voting |

**Cumulative:** All Wave 1-4 features + Wave 5 features

## Plan

### Goals
Implement private poll access control with cryptographic invites.

### Planned Deliverables
- [ ] Private visibility mode for polls
- [ ] PollInvite record generation
- [ ] Invite link creation with expiration
- [ ] Access validation before voting
- [ ] Invite management dashboard
- [ ] Bulk invite generation
- [ ] Invite usage tracking
- [ ] Revoke invite functionality

### Success Metrics
- Private polls only accessible to invited users
- Invite records work correctly on-chain
- Creators can manage invites easily
- Uninvited users cannot access private polls
